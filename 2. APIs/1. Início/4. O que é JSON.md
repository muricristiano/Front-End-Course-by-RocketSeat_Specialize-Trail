# O que é JSON

É um formato de arquivo leve para troca de dados e informações

Fácil leitura humana

Fácil tratamento de dados por sistemas

Universal, independente de linguagem (Java, C#, JS, etc... )


- Documentação JSON:
https://json.org/json-en.html
